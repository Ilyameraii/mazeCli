# Maze CLI

Консольная игра-лабиринт, разработанная на C# и .NET.

При каждом запуске создаётся новый случайный лабиринт. Игрок должен найти путь от стартовой позиции до конечной точки, управляя персонажем с помощью клавиш со стрелками.

Проект демонстрирует алгоритмическую генерацию лабиринтов, работу с консольной графикой и применение объектно-ориентированного подхода.

## Возможности

- Случайная генерация лабиринта при каждом запуске
- Управление персонажем с клавиатуры
- Автоматический поиск наиболее удалённой точки для размещения финиша
- Цветная визуализация в консоли
- Проверка столкновений со стенами
- Экран завершения игры

## Архитектура проекта

### Program

Точка входа приложения.

Создаёт игровой объект и запускает игровой цикл.

### MazeGame

Основной игровой контроллер.

Отвечает за:

- создание лабиринта;
- обработку ввода;
- проверку условий победы;
- взаимодействие между игровыми объектами.

### MapGenerator

Генератор лабиринта.

Выполняет:

- построение карты;
- создание проходов;
- определение самой удалённой клетки;
- установку точки финиша.

### MapRenderer

Отрисовка игрового поля в консоли.

Функции:

- отображение карты;
- отображение игрока;
- отображение финиша;
- обновление экрана при движении.

### Player

Модель игрока.

Содержит:

```csharp
public int PosX;
public int PosY;
public char Skin;
```

Обрабатывает перемещение по карте.

## Алгоритм генерации лабиринта

Для создания карты используется алгоритм поиска в глубину (Depth-First Search).

Этапы работы:

1. Карта полностью заполняется стенами.
2. Выбирается стартовая клетка.
3. Случайным образом выбирается направление движения.
4. Между клетками создаются проходы.
5. Процесс продолжается до обхода всех доступных клеток.
6. Самая удалённая клетка назначается финишем.

Схема:

```text
Старт
  ↓
Создание стен
  ↓
DFS-обход
  ↓
Формирование проходов
  ↓
Поиск дальней точки
  ↓
Установка финиша
  ↓
Готовый лабиринт
```

## Управление

| Клавиша | Действие |
|----------|----------|
| ↑ | Вверх |
| ↓ | Вниз |
| ← | Влево |
| → | Вправо |
